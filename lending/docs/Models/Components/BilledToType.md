# BilledToType

Defines if the invoice or credit note is billed/rebilled to a project or customer.

## Example Usage

```csharp
using Codat.Lending.Models.Components;

var value = BilledToType.Unknown;
```


## Values

| Name            | Value           |
| --------------- | --------------- |
| `Unknown`       | Unknown         |
| `NotApplicable` | NotApplicable   |
| `Customer`      | Customer        |
| `Project`       | Project         |
